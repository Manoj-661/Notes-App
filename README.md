📝 Notes App

A simple and responsive web-based Notes Application that allows users to create, view, edit, and delete notes easily.
This project was developed to practice full-stack web development concepts, including CRUD operations and frontend–backend integration.

The application provides a clean and intuitive interface for managing personal notes efficiently.


🚀 Project Overview

The Notes App is designed to help users store and manage notes digitally. Users can quickly add new notes, update existing notes, and remove notes when they are no longer needed.

This project demonstrates the implementation of:

CRUD Operations (Create, Read, Update, Delete)
Client–Server communication
Basic full-stack project structure
Version control using Git and GitHub.


✨ Features

✔ Create new notes
✔ View saved notes
✔ Edit existing notes
✔ Delete notes
✔ Simple and user-friendly interface
✔ Lightweight and fast application

⚙️ Installation & Setup

Follow these steps to run the project locally.

1️⃣ Clone the repository
git clone https://github.com/Manoj-661/Notes-App.git

2️⃣ Navigate to the project directory
cd Notes-App

3️⃣ Install dependencies
npm install


🧪 .env Setup
Backend (/backend)

MONGO_URI=<your_mongo_uri>

UPSTASH_REDIS_REST_URL=<your_redis_rest_url>
UPSTASH_REDIS_REST_TOKEN=<your_redis_rest_token>

NODE_ENV=development


🔧 Run the Backend
cd backend
npm install
npm run dev


💻 Run the Frontend
cd frontend
npm install
npm run dev
